
Комментарии для игроков
-----------------------

1. Регистр букв в коде не имеет значения (по крайней мере, для русских и английских букв).

2. Код не может содержать пробелов, а также следующих символов: `'",!<>

3. Счетчики времени до подсказки и времени на задании имеют погрешность 1-2 секунды.
На статистику это не влияет.

4. Автоматическое обновление (а также обновление по ссылке "обновить сейчас")
сначала проверяет наличие связи с интернетом, а только потом обновляет движок.
Поэтому обновлять через ссылку "обновить сейчас" безопаснее, чем по F5 -
нет риска получить пустую страницу при отвалившемся интернете.

5. По завершению времени на задание, а также при наступлении времени полученя подсказки
обновление происходит автоматически (разумеется, при наличии связи с интернетом).
